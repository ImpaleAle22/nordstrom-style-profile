# Session-Based Personas Archive

This directory contains the archived session-based personas demo that attempted to show profile evolution over time through session scrubbing.

## Archived Files
- `page.tsx` - Server component that fetches profiles
- `PersonaDemoUI.tsx` - Client component with session fetching and scrubbing logic
- `DemoControls.tsx` - Sidebar with session navigation controls

## Why Archived
The session scrubbing feature proved too complex to get working reliably and would require significant data work to tell good visual stories. The simplified version (just showing current state) is more practical for the demo.
