This page demonstrates the account-nav.js and site-header.js components. All icons are official Nordstrom SVGs loaded from the Iconography/ folder.

How It Works

The header and sidebar navigation are rendered from JavaScript modules, not inline HTML. This means you can:

  • Update icons in one place across all pages
  • Customize user data (name, store, rewards) dynamically
  • Maintain consistency across the entire site
  • Easily modify navigation structure without touching HTML

Usage Example

<!-- In your HTML -->
<div id="site-header-container"></div>
<div id="main-nav-container"></div>
<div id="account-nav-container"></div>

<!-- In your script -->
<script type="module">
  import { initSiteHeader } from './components/site-header.js';
  import { initAccountNav } from './components/account-nav.js';

  initSiteHeader({
    userName: 'Brian',
    storeName: 'Michigan Avenue'
  });

  initAccountNav({
    activeItem: 'your-style',
    userName: 'Brian'
  });
</script>
